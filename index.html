ABSTRACT

Text classification and sentiment analysis are challenging tasks, which have raised a high interest in the Machine learning community. In this PROJECT we perform sentiment analysis on a twitter dataset. First, optimization of text preprocessing techniques were performed and new features were added to the dataset. Then we implemented a XBoost algorithm and six Neural Networks models, exploring the performance of LSTM, bidirectional LSTM, CNN and CNN+LSTM frameworks. We assessed different approaches to training embeddings as well, which leads us to conclude that the best approach uses GloVe embeddings with CNN+LSTM.
